\doxysubsection{Detailed Description}
Represents a movable object in the game world. 

\doxylink{class_entity}{Entity} adds velocity and movement capabilities to \doxylink{class_scene_node}{Scene\+Node} objects. It provides methods for setting and modifying velocity, as well as updating the entity\textquotesingle{}s position based on elapsed time.
