\doxysubsection{Detailed Description}
\doxylink{class_state}{State} stack manager for handling game state transitions. 

Manages a stack of game states with LIFO semantics, providing\+:
\begin{DoxyItemize}
\item \doxylink{class_state}{State} transitions (push/pop/clear)
\item Update/draw propagation
\item Input event routing
\item \doxylink{class_state}{State} factory registration 
\end{DoxyItemize}
