\doxysubsection{Detailed Description}
Bitmask categories for scene object classification and collision filtering. 

These categories are used to\+:
\begin{DoxyItemize}
\item Manage object interactions (collision detection)
\item Filter rendering operations
\item Organize scene graph queries
\end{DoxyItemize}
