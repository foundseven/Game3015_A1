\doxysubsection{Detailed Description}
Main menu state handling title screen presentation and navigation. 

Manages core menu functionality including\+:
\begin{DoxyItemize}
\item Animated menu elements
\item Menu option selection
\item Transition to gameplay states
\item Visual effects (e.\+g., ship sprite animation)
