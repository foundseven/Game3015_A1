Detailed Description

Manages player input and actions.

Handles player input and maps actions to commands.

The Player class handles key bindings, real-time input, and action execution for the player.

The Player class manages key bindings and translates user input into game commands that are pushed into a CommandQueue. It supports both event-driven and real-time input handling.
